# Task Manager Application

A Python-based command-line Task Manager application that helps users organize, manage, and track tasks efficiently. This project demonstrates file handling, data management, user interaction, and modular programming principles.

---

## Features
- Add, update, delete, and view tasks
- Mark tasks as completed or pending
- Categorize tasks by status and priority
- Save tasks persistently using file storage
- Search tasks by keywords
- Display summary and detailed task reports

---

## How to Use
1. Download or clone the repository.
2. Ensure you have Python 3 installed on your computer
